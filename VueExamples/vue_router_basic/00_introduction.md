## 介绍

Vue Router是Vue.js的官方路由。它与Vue.js核心深度集成，让用Vue.js构建单页应用变得轻而易举。功能包括：

* 嵌套路由映射
* 动态路由选择
* 模块化、基于组件的路由配置
* 路由参数、查询、通配符
* 展示由Vue.js的过渡系统提供的过渡效果
* 细致的导航控制
* 自动激活CSS类的链接
* HTML5 history模式或hash模式
* 可定制的滚动行为
* URL的正确编码
