Vue Transition Type 属性示例

本示例展示了当元素同时具有CSS动画(2秒)和过渡(5秒)时，type属性如何影响Vue监听的事件类型：

1. 无type属性 - 默认行为

Vue会自动检测并监听持续时间较短的动画事件(这里是animation)

状态: {{ showDefault ? '显示' : '隐藏' }}

2. type="transition"

Vue只监听transitionend事件，忽略animationend事件(过渡持续5秒，动画持续2秒)

状态: {{ showTransition ? '显示' : '隐藏' }}

3. type="animation"

Vue只监听animationend事件，忽略transitionend事件

状态: {{ showAnimation ? '显示' : '隐藏' }}
