y=5，下面的表格解释了这些算术运算符：

### 用于字符串的+运算符
+运算符用于把文本值或字符串变量加起来（连接起来）。

```js
// 实例
txt1="What a very";
txt2="nice day";
txt3=txt1+txt2;

// txt3运算结果如下: What a verynice day

txt3=txt1+" "+txt2;

// txt3运算结果如下: What a very nice day
```

### 对字符串和数字进行加法运算
两个数字相加，返回数字相加的和，如果数字与字符串相加，返回字符串，如下实例：

```js
// 实例
x=5+5;
y="5"+5;
z="Hello"+5;
// x,y, 和 z 输出结果为:10 55 Hello5

// 规则:如果把数字与字符串相加，结果将成为字符串！
```
