# 掌上外研 - 登录注册页面

基于 MasterGo 设计图实现的精美登录注册页面，包含丰富的动画效果和交互体验。

## 📁 文件结构

```
html_examples/

├── login.html                  # 主 HTML 文件
├── login.css                   # 样式文件
├── login.js                    # 交互逻辑
├── README.md                   # 说明文档
└── UPDATE_LOG.md              # 更新日志
```

## ⚠️ 重要提示

**书籍装饰图片**：需要将书籍装饰图片保存为 `assets/books-decoration.png`。详细说明请查看 [assets/README_ASSETS.md](assets/README_ASSETS.md)。

## ✨ 特性

### 视觉效果
- **3D 书籍堆叠动画** - 立体感强的书籍装饰，带有浮动动画
- **铅笔装饰** - 旋转的铅笔元素,增添趣味性
- **光效动画** - 柔和的呼吸光效，提升视觉层次
- **渐变背景** - 多层次的颜色渐变，营造温暖氛围
- **黄色滑块** - 标签切换时的流畅滑动指示器

### 交互体验
- **标签切换** - 密码登录/验证码登录平滑切换，带黄色滑块动画
- **验证码功能** - 验证码发送、60秒倒计时
- **输入框动效** - 聚焦时的抬升和阴影效果
- **按钮动画** - 悬停时的光泽扫过效果
- **复选框动画** - 选中时的弹跳和对勾绘制动画
- **链接下划线** - 悬停时的下划线滑入效果
- **消息提示** - 优雅的 Toast 消息提示

### 设计细节
- **精确还原设计稿** - 严格按照 MasterGo DSL 数据实现
- **CSS 变量管理** - 统一的颜色和样式管理
- **流畅的过渡** - 所有交互都采用缓动函数
- **无障碍支持** - 语义化 HTML 和键盘导航

## 🚀 使用方法

### 1. 直接打开
```bash
# 在浏览器中打开
open login.html
# 或
start login.html
```

### 2. 本地服务器
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js 的 http-server
npx http-server -p 8000
```

然后访问: `http://localhost:8000/login.html`

## 🔧 自定义配置

### 修改主题色
在 `login.css` 的 `:root` 中修改 CSS 变量:

```css
:root {
    --color-primary: #FF7F00;        /* 主题色 */
    --color-primary-light: #FFE79C;  /* 浅色主题 */
    /* ... 其他变量 */
}
```

### 调整动画速度
修改相关 CSS 属性:

```css
.decoration-books {
    animation: float 3s ease-in-out infinite;  /* 浮动速度 */
}

.form-input {
    transition: all 0.3s;  /* 输入框过渡时间 */
}
```

### 禁用动画
在 CSS 文件顶部添加:

```css
* {
    animation: none !important;
    transition: none !important;
}
```

## 📝 表单验证

### 手机号验证
- 必须为 11 位数字
- 必须以 1 开头
- 第二位为 3-9

### 密码验证
- 最少 6 位字符
- 支持字母、数字、特殊字符

### 验证码验证
- 必须为 6 位数字
- 60 秒倒计时限制重发

## 🎬 交互说明

### 标签切换
点击"密码登录"或"验证码登录"标签时：
1. 黄色滑块流畅移动到对应标签下方
2. 表单字段平滑切换（淡入淡出效果）
3. 密码登录模式显示密码输入框和"记住密码"选项
4. 验证码登录模式显示验证码输入框和"发送验证码"按钮

### 验证码发送
1. 点击"发送验证码"按钮
2. 验证手机号格式
3. 显示"验证码已发送"提示
4. 按钮进入 60 秒倒计时状态
5. 倒计时期间按钮禁用
6. 倒计时结束后可再次发送

### 消息提示
- 成功消息：绿色背景
- 错误消息：红色背景
- 自动 3 秒后消失
- 带有淡入淡出动画
